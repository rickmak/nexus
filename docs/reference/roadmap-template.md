# Roadmap Template

Use this template for each new planning cycle (quarterly or monthly).

---

## How to use this file

- Keep roadmap entries short; link to detailed plans/issues.
- Update item status when work starts, blocks, lands, or is dropped.
- Keep `Current iteration` to active priorities only.

### Status legend

- `proposed`: idea captured, not yet accepted
- `planned`: accepted and queued
- `in_progress`: active implementation
- `blocked`: active but waiting on dependency/decision
- `done`: completed and verified
- `dropped`: intentionally de-scoped
